A multi-agent AI assistant that drafts support tickets, academic papers, professional letters, and
college applications. Built on a decoupled React SPA and TypeScript/Express backend hosting five
specialized agents powered by Google Gemini and OpenAI. Features live web RAG via Tavily, Scalekit
OIDC authentication, JWT-signed sessions, and strict Zod validation.
